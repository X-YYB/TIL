# 热点面试题
## 1、==和equals区别？
答：==比较的是引用的地址，equals比较的是字符串里面的内容，一般情况下，判断字符串是否为空，会选择equals。
## 2、String、StringBuffer、StringBuilder的区别？
答：String是不可变对象，线程安全，效率低。
       StringBuffer是可变对象，线程不安全，效率中等。
       StringBuilder也是可变对象，线程不安全，效率最高。
